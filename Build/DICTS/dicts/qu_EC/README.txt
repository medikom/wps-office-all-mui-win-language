-------------------------------------------------------------
 NB! Before ooo 3.2 there is no language list box entry for Kichwa
-------------------------------------------------------------
This version is an early release, and is by no means complete.

From version 1.0 and on, twofold suffix stripping is used. This greatly reduces the size of the package, and the complexity of the generation. 
The 170000+ words .dic was reduced to about 3000 words.

From version 1.0 there is also a basic hyphenator included.


-------------------------------------------------- -----------
 ¡NÓTESE BIEN! Antes de ooo 3.2 no había ninguna entrada en el cuadro de lista de idiomas para kichwa
-------------------------------------------------- -----------
Esta versión es una versión preliminar y de ninguna manera está completa.

A partir de la versión 1.0 en adelante, se utiliza la eliminación de sufijos dobles. Esto reduce en gran medida el tamaño del paquete y la complejidad de la generación.
Las más de 170.000 palabras .dic se redujeron a unas 3.000 palabras.

A partir de la versión 1.0 también se incluye un separador de palabras básico.
